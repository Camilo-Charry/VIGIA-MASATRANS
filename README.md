# VIGIA-MASATRANS
Sistema Inteligente de Gestión HSEQ y Talento Humano
1. OBJETIVO DEL PROYECTO

Crear un sistema local empresarial para MASATRANS que permita:

Gestionar empleados.
Controlar vencimientos de cursos y documentos.
Generar alertas.
Tener seguimiento HSEQ.
Reemplazar la matriz Excel actual.
Mejorar la organización y visualización de información.
Facilitar auditorías y controles.

El sistema será usado inicialmente en:

Área GH.
Aproximadamente 5 computadores.
Red local interna.
2. NOMBRE DEL SOFTWARE
VIGIA MASATRANS

Concepto:

“VIGIA” representa vigilancia, control y prevención.
Profesional.
Fácil de recordar.
Suena corporativo.
